# Docs — index

This directory contains reports, issue summaries, and technical support documentation for the repository.

## What belongs here

- issue reports
- repository audits
- decision-record guidance
- documentation migration notes
- other technical reports that are not project-memory files

## What does not belong here

`docs/` does **not** replace:
- `.agent/memory/current_state.md`
- `.agent/memory/domain_map.md`
- `ROADMAP.md`

Those files remain the owners of current implementation truth and planning scope.
